
# 🏥 Medical Waste Prediction System

This project was developed as part of our university final-year group project. The goal is to predict the amount of medical waste generated by healthcare facilities using machine learning techniques. Accurate prediction can help hospitals and healthcare authorities improve waste collection, resource planning, and disposal management.

## Project Overview

Medical waste is increasing with the growth of healthcare services. Managing this waste efficiently is important for protecting public health and the environment. In this project, we used a synthetic healthcare dataset and compared multiple machine learning regression models to identify the one that provides the most accurate predictions.

## Features

- Predicts medical waste generation
- Uses synthetic healthcare data for training
- Compares multiple machine learning regression models
- Evaluates model performance using standard metrics
- Identifies the best-performing model for forecasting

## Technologies Used

## Machine Learning Models

- Gradient Boosting Regressor
- XGBoost Regressor
- Random Forest Regressor
- Extra Trees Regressor
- AdaBoost Regressor
- Hist Gradient Boosting Regressor
- Voting Regressor
- Stacking Regressor

## Results

After comparing all models, the **Gradient Boosting Regressor** achieved the best performance.

- **R² Score:** 0.9565
- **MAE:** 3391.13
- **RMSE:** 5879.39
- **MAPE:** 9.48%
